# Projeto E-commerce API - Estudo e Prática Profissional

Este repositório contém o desenvolvimento de uma API REST para um sistema de e-commerce, focada em fundamentos sólidos de backend. O projeto faz parte da minha jornada de aprendizagem contínua em tecnologia.

## 🚀 Objetivo do Projeto
Este é um projeto de estudo e aprendizagem que se conecta diretamente com minhas práticas profissionais. Aprofundar-me nestes conceitos introdutórios me permite aperfeiçoar a qualidade das minhas entregas e aumentar minha eficiência no ambiente de trabalho, garantindo código mais limpo, seguro e performático.

## 🛠️ Tecnologias e Conceitos Aplicados
Conforme o cronograma de estudos do curso da RocketSeat,este projeto abrange:

*   **Fundamentos do Python:** Estruturas de dados e lógica aplicada.
*   **Conceitos de API:** Arquitetura RESTful e comunicação via JSON.
*   **Métodos HTTP:** Implementação de GET, POST, PUT e DELETE.
*   **Banco de Dados:** Utilização de SQL e ORM (SQLAlchemy) para persistência de dados.
*   **Flask:** Micro-framework para criação de rotas e lógica de servidor.
*   **Autenticação com Flask-Login:** Gestão de sessões e segurança de usuários.
*   **Tratamento de Erros:** Respostas HTTP adequadas e validações de dados.

## 📂 Como executar
1. Instale as dependências: `pip install flask flask_sqlalchemy flask_cors flask_login`
2. Execute o servidor: `python app.py`
3. Acesse em: `http://127.0.0.1:5000`
